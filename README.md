## About Project

### Purpose
=> 수강 신청,교재 판매 수익 사이트
<br>
=> 배우고 싶은 강의에 대한 쉬운 접근성
<br>
=> 멘토링을 통한 질문 답변 시스템
<br>
<br>

**개발 기간** : 2023.03 ~ 2023.04 (2주)
<br>
**개발 인원** : 7명
<br>
**담당 역할** : 멘토링 게시판, 대화방 목록 및 대화방 기능 구현 및 프로젝트 테스팅
<br>
**플랫폼** : WEB
<br>
<br>
### Improvements

- **예약 가능 항목** : <br>같은 시간대에 예약이 이미 차있는 경우 예약을 막아야 합니다. 

- **예약 알림** : <br>예약 승인이 되거나 거부가 된다면 지원자에게 알림을 제공해야 합니다.

- **실시간 대화** : <br>http 요청에 의한 단방향 통신 방식 말고 실시간 대화가 가능한 양방향 통신을 구현해보고 싶습니다. 
<br>
<br>

### Review
<br>
많은 변경 사항으로 인해 초기 설계와 달라진 부분이 많았습니다.<br>
이에 따라 설계 수정에 상당한 시간을 투자했고, 따라서 결과물이 많이 아쉬웠습니다. <br>
특히, 양방향 통신을 지원하는 WebSocket을 활용한 채팅 시스템을 구현해보고 싶었지만 시간적 제약이라는 어려움 때문에 미쳐 구현하지 못했습니다. <br>
사이드 프로젝트로 WebSocket을 활용하는 예제를 꼭 시도해봐야겠다고 생각했습니다.
